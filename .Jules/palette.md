## 2024-11-20 - Semantic Buttons for Tabs
**Learning:** Converting div-based tabs to semantic `<button>` elements requires careful CSS resets (background, border, font-family) to maintain the original design while gaining native keyboard accessibility.
**Action:** Always check `background`, `border`, `padding`, and `font` properties when replacing interactive `div`s with `button`s to avoid browser default styling regressions.
